# Prime Numbers

## Description

## Instructions

1. Install [Node.js](https://nodejs.org/en/download/)
2. Clone the repository

  `$ git clone https://github.com/acucciniello/prime-numbers.git`

3. Enter the project directory

  `$ cd prime-numbers `

4. Run the Script

  `$ node prime-numbers.js`
